MPI Oriented Development and Investigation or MODI provides a cluster of 8 compute nodes that can be utilized for long running batch jobs.
The system is configured as a SLURM cluster in which regular slurm commands can be utilized.
This includes functionality such as 'sbatch', 'squeue', 'sinfo' and friends.

The current setup does not at the moment support interactive functionalities such as 'srun' and similar.

Please refer to the separate MODI user guide for details about using MODI. In particular the part about running 'sbatch' commands using the shared modi mount directory is important for jobs to succeed.
